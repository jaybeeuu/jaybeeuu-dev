Word EDF
Office integration into a legacy web application.

Word EDF

Making use of the VSTO tool-set, ASP and some COM+ objects the Word EDF project aimed to allow users to view and upload letters created based on a template with a minimal client footprint.

Taking a lead role in the development of this project I was responsible for writing the ASP pages used to start the process, the C++ COM objects used to create the documents, the Word tool-bar ribbon forming the office integration and then the ASP pages used to receive the final document.

Working on this project gave me an insight into the inner workings of a web based application and the ways in which it communicates with the ecosystem of supporting applications around it.
